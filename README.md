# Servo Motors Task

## Description
This project controls 4 servo motors using Arduino.

### Required Behavior
- Run all 4 servo motors in Sweep mode for 2 seconds.
- After 2 seconds, all servo motors stop and hold at 90°.

## Files Included
- 4servo.ino → Arduino source code.
- 4ser.mp4 → Video showing the circuit wiring and operation.
